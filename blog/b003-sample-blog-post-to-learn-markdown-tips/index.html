Getting Started#

This is a demo post showcasing how to write blogs using Markdown. I was inspired by this article and highly recommend spending just 5 mins to learn markdown. It is a powerful way to turn plain text into bold, italics, tables, and much more.

For people who are interested to learn some some more advanced tips about using markdown using Jekyll click here

you can visit this Markdown Tutorial to learn and practice to try out, but below are some quick glapses.

Basics#

Italic#

italic surround words with an ( _ );

Bold#

Bold surround words with two asterisks ( ** );

italic and bold#

You can to make the text italic and bold by combine ** and _, **italic and bold** sourrnd with (**_ );

List#

Unordered List and Ordered List

Unordered List#

Using asterisk ( * ) to create lists, I added one * in front of each words.

- apple
- pear
- milk
- eggs

Ordered List#

Using numbers instead:

  1. Red
  2. Green
  3. Blue

Header#

Using hash mark ( # ) to create headings,

- # Header one

- ## Header two

- ### Header three

- #### Header four

- ##### Header five

Links#

  [How to add inline links](https://www.markdowntutorial.com/lesson/3/):
  wrap link text in brackets[], and wrap link in parentheses ().

How to add inline links

Links to Heading#

Wrap links into headings: #[link text here in this brackets ](actual links in this parentheses), depends on how big you want the heading, and add relavent numbers of hash mark in front of[], you can try it out here.

Paragraghs with Togetherness#

Applying two spaces after each new line to create new para.

Two spaces after each new line:

When you are old and grey and full of sleep,
And nodding by the fire, take down this book,
And slowly read, and dream of the soft look
Your eyes had once, and of their shadows deep;

  If there is no space the poetry will be look like one paragragh without stop, see below:

When you are old and grey and full of sleep, And nodding by the fire, take down this book, And slowly read, and dream of the soft look Your eyes had once, and of their shadows deep;

Table#

Using Pipe Symbol (|) and Header Separator (—)to create tables.

  Use | to separate columns.
  Use --- (with optional : for alignment) to define the header.
  You can add as many rows and columns as you need.
Number Next number Previous number
Five Six Four
Ten Eleven Nine
Seven Eight Six
Two Three One
